部署完成后，手机用 **4G/5G 或任意 WiFi** 都能打开，并可「添加到主屏幕」当 App 用。

线上地址（部署成功后）：

**https://xiaying-xia.github.io/life-effort-viz/**

## 方式 A：GitHub Actions 自动部署

> 需要 Token 勾选 **workflow** 权限。若不想改 Token，请用下面的方式 B。

（已默认采用方式 B）

## 方式 B：本地一键发布（gh-pages 分支，推荐）

在 Mac 终端：

```bash
cd ~/Projects/life-effort-viz
npm run deploy:pages
```

然后在 GitHub 仓库 **Settings → Pages**：
- Source：**Deploy from a branch**
- Branch：**gh-pages** / **/ (root)**

## 方式 C：GitHub Actions（可选，需 workflow 权限）

### 第 1 步：创建 GitHub 仓库

1. 打开 https://github.com/new
2. 仓库名填 **`life-effort-viz`**
3. 选 **Public**
4. 不要勾选 README（本地已有）
5. 点 **Create repository**

### 第 2 步：推送代码

```bash
cd ~/Projects/life-effort-viz
git init
git add .
git commit -m "Initial commit: 人生努力可视化 PWA"
git branch -M main
git remote add origin https://github.com/xiaying-xia/life-effort-viz.git
git push -u origin main
```

若提示用户名密码：
- Username：`xiaying-xia`
- Password：GitHub Personal Access Token（`ghp_` 开头，勾选 **repo**）

### 第 3 步：开启 GitHub Pages

1. 仓库 → **Settings** → **Pages**
2. **Build and deployment** → Source 选 **GitHub Actions**
3. 推送代码后，Actions 会自动构建部署（约 1～2 分钟）
4. 在 **Actions** 标签页查看进度，成功后 Pages 设置页会显示网址

### 第 4 步：手机安装

1. Safari / Chrome 打开：**https://xiaying-xia.github.io/life-effort-viz/**
2. iPhone：分享 → **添加到主屏幕**
3. Android：菜单 → **安装应用**

## 以后更新

改完代码后：

```bash
git add .
git commit -m "更新功能"
git push
```

GitHub Actions 会自动重新部署，手机刷新或重新打开即可。

## 常见问题

**Q：打开是 404？**  
等 1～2 分钟再试；确认 Pages 已开启且 Actions 构建成功。

**Q：PWA 安装按钮没出现？**  
必须用 **https** 打开（GitHub Pages 默认就是 https）。

**Q：数据和电脑不一样？**  
数据存在各自浏览器本地，不同设备不自动同步（后续可加云同步）。
